Horizontal Scrollbar in Markdown Tables Programming

I'm using the JavaScript library Marked for rendering Markdown content. By default, the rendered tables do not have the overfolow-x: auto property. So, if tables get too large they may mess up the styling. An easy fix can be to automatically use a horizontal scrollbar by setting the overflow style to overfolow-x: auto. The Marked library is pretty flexible and allows you to customize how elements are rendered. You can submit a custom renederer for tables as follows:

 const renderer = {
      table(header, body) {
          if (body) body = `<tbody>${body}</tbody>`;
          return `<div style="overflow-x:auto">
              <table>
              <thead>
              ${header}
              </thead>
              ${body}
              </table>
              </div>`;
      },
  };
  marked.use({ renderer });
  
  let text = "...SOME MARKDOWN WITH TABLES...";
  let source = marked.parse(text);

For more information about how to customize Marked further visit the Extensibility section of the documentation.
